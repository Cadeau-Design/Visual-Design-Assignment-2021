Christ, the Redeemer and the wave pattern as an animated GIF.


I have created the GIFs published on this page as follow, first I have saved all images that I have previously manipulated in two different folders, one for each GIF and then imported them into Photoshop by following the steps explained below:

1. Load images into Photoshop: File>Script>Load Files into Stock

2. Open the timeline window

3. Create frame animation

4. Give each frame a layer

5. Make frames from layers

6. Decide on the frame's durations (the images with text appear longer on the screen)

7. Decided on the looping (always)

8. Save for the web as GIF

Final file (1) information: exported as GIF, dimensions: 1080 x 1080 pixels, GIF 128 Dithered (gives a dotted texture to the image) and 256 Colours and size:196 KB.

Final file (2) information: exported as PNG-24, dimensions: 600 x 249 pixels, GIF 128 Dithered (gives a dotted texture to the image) and 256 Colours and size:323 KB.
